jsOAuth plugin for Google Chrome Extension

Evernote JavaScript Sample with Basic Access Permissions

jsOAuth is a JavaScript implimentation of the OAuth protocol. Currently supports version 1.0 (RFC5849) of the specification.

The jsOAuth plugin for Google Chrome Extension is an extension that enables Google Chrome Extension support. Please checkout README for full description.

You'll need to create an account on Evernote development server. And then "Get An API Key" and fill in the consumerKey and consumerSecret in the form below. (Normally you will put these keys directly somewhere in your code.)

You can revoke access authorization on Applications management page. If you did that, refresh this page to re-enable the "Authroize" button.

Please report issues here.
